เข้าหน้า User Management

  • 1

  • 2

3
เลือก Scope (ขอบเขตข้อมูลที่เห็น)
กำหนดว่า Vendor คนนี้เห็นข้อมูลของบริษัทใหนได้บ้าง

ขอบเขตการเข้าถึง Vendor (Scope) มี 3 แบบ:

🔵 เฉพาะ Vendor ตัวเอง (ปกติ)
เห็นเฉพาะข้อมูลของบริษัทตัวเองเท่านั้น — ใช้สำหรับเซลล์/ตัวแทนทั่วไป
🟡 เลือก Vendor เฉพาะ
เห็นข้อมูลของหลายบริษัทที่กำหนดไว้ — ใช้สำหรับตัวแทนที่ดูแลหลายสาขา
🟢 Parent + ทุก Vendor ในกลุ่ม
เห็นทุก Vendor ในกลุ่มอัตโนมัติ — ใช้สำหรับ Admin ของบริษัทแม่
💡 แนะนำสำหรับ Vendor ทั่วไป

เลือก "เฉพาะ Vendor ตัวเอง" ไว้ก่อนเสมอ ปลอดภัยที่สุดครับ

4
ยืนยันผลลัพธ์
ตรวจสอบว่าบัญชี Vendor ถูกสร้างสำเร็จ
  • ✓
    จำนวน Vendor เพิ่มขึ้น ใน Stats Card (จาก 1 → 2)
  • ✓
    รายชื่อใหม่ขึ้นพร้อม badge สีเขียว "Vendor"
  • ✓
    คอลัมน์ VENDOR แสดงชื่อบริษัทที่ผูกไว้ถูกต้อง
  • ✓
    สถานะแสดงเป็น เปิดใช้
